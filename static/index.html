About

Welcome to the dragon depiction map. This site combine information from OpenStreetMap and Wikidata about where you can find things depicting dragons across the world.

This is an open-source software. See sources

Contribute

Data is refreshed every three hours for now. Your changes may take a while to appear.

In case the data is correct on those website, but are badly adapted for this site, I can add overrides for some values. E-mail me at dragon@mariusdavid.fr. You can also send image of the work which does not fit the accepted license on those sites. I’ll add them here (with credit to the image author).

OpenStreetMap

You need to use the artwork_subject key with a value containing “dragon” (case-insensitive). Note that code for now remove any result that contain “dragonfl” to remove things depicting dragonflies.

More possible keys may be found on the wiki. Note that subject:wikidata is currently ignored.

Wikidata

For an item to appear on this map, it need to fullfill those two conditions:

  • Some form of geographical information
    • Either some coordinate (P625)
    • or a place (P276) which have coordinate (or is a part of a place which have coordinate)
  • It needs to depict (P180) a dragon (Q7559) (or dragon in fictional work, Q30170627) or one of its subclass/an object that have a nature of dragon
